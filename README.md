# 🛒 WarungKu

WarungKu adalah aplikasi sederhana untuk membantu mencatat stok barang di warung.

## Fitur

- ✅ Tambah Barang
- ✅ Edit Barang
- ✅ Hapus Barang
- ✅ Total Barang
- ✅ Validasi Input
- ✅ Enter Keyboard
- ✅ Auto Focus

## Teknologi

- HTML
- CSS
- JavaScript

## Status

🚧 Beta Development

## Tujuan

Project ini dibuat sebagai latihan JavaScript sekaligus untuk membantu pencatatan stok warung agar lebih praktis.
