# 🛒 WarungKu

Aplikasi inventori sederhana berbasis JavaScript untuk mengelola stok barang.

## ✨ Fitur

- ✅ Menambah barang
- ✅ Mengedit barang
- ✅ Menghapus barang
- ✅ Pencarian barang
- ✅ Dashboard statistik
- ✅ Status stok (Normal, Menipis, Habis)
- ✅ Export CSV
- ✅ Local Storage

## 🛠️ Teknologi

- HTML
- CSS
- JavaScript
- LocalStorage API

## 🚀 Cara Menjalankan

1. Download atau clone repository.
2. Buka `index.html` di browser.
3. Mulai kelola inventori.
